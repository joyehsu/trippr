隱私權政策

最後更新日期：2026年3月24日

歡迎使用 Trippr（以下簡稱「本應用程式」）。本隱私權政策旨在說明我們如何收集、使用、保護及處理您的個人資訊。

1. 資訊收集

當您使用本應用程式時，我們可能會收集以下資訊：

  • 在使用 AI 行程解析功能時您所輸入的文字資訊。
  • 設備基本資訊（如瀏覽器類型、語言設定）。

2. 資訊使用

我們收集資訊的主要目的是為了提供並改善本應用程式的服務，特別是透過 Google Gemini API 提供行程建議與解析功能。

3. 關鍵技術與第三方服務

本應用程式使用 Google Gemini API 服務。您輸入的部分資訊可能會傳遞至 Google 進行處理。請參考 Google 隱私權政策 以了解更多詳情。

4. 資料安全

我們採取合理的安全措施來保護您的資訊，防止未經授權的存取或洩漏。

5. 聯絡我們

如果您對本隱私權政策有任何疑問，請透過 GitHub 專案頁面與我們聯繫。
